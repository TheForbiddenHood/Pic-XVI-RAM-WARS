# PIC XVI: RAM WARS
This repo is for the Microcomputer Systems I course at UT San Antonio. This serves as a housing for the first iteration of the RTS game in a more centralized work environment.

This game focuses on creating a small-scale RTS multiplayer experience using two PIC16s and ATmegas to allow players to compete using their efficiency at micro-management of their resource-collection, army-scaling, and research into military upgrades to bring them success in the fast-paced battle phase system. First player to win best out of three battles wins!

## Why is this repo public?
I want this to not only be available to team members on the project, but also be visible for others for presentation as well as a resource to learn from. This program being in the Arduino IDE is entirely in C++, and is fairly easy to just download and press *Open Project* in your IDE. Just as simple as this game is to pick up and play, so should this project be to learn from!

I'm trying my best to pro-actively make changes and also apply more comments to ensure it is easy to follow what is going on throughout the code - however, I do understand it can be a bit of a mouthful at time please feel free to reach out and ask me any questions! C:
